Pen: Sticky Slider Nav

by Ettrics

The author of this Pen did a great job creating a very professional looking website which incorporates all kinds of different features I recognize and many I don't. They added a beautiful navigation bar that moves with the user along the page and links you to each different section of the page. The following are some of what they used:

  • flexbox
  • font styling
  • media box
  • et-slide
  • et-hero-tab
  • Various JS code
